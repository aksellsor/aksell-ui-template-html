## Styles

Base styles are loaded from [@aksell/ui](https://www.npmjs.com/package/@aksell/ui) via CDN.
Add project-specific styles in `style.css`.
Do not add Tailwind, Bootstrap, or another CSS framework.
`@aksell/ui` does not ship Aeonik font files. If this project has a valid Aeonik license, add the project-owned `@font-face` declarations in `style.css`.

## Using @aksell/ui components

Add CDN links in `index.html` and import standalone JS modules in `main.js`.
See `AGENTS.md` for available exports.
